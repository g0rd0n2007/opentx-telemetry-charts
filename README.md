## OpenTX Telemetry Charts
Just change first lines to configure the script.

Name of Your sensor:
- sensorName = "Lipo"

Value format. In this case float with 2 digits after dot:
- valueFormat = "%.2f"

Value unit:
- valueUnit = "[V]"
